## remove node

Write a function, _removeNode_, that takes in the head of a linked list and a target value as
arguments. The function should delete the node containing the target value from the linked list and
return the head of the resulting linked list. If the target appears multiple times in the linked
list, only remove the first instance of the target in the list.

Do this **in-place**.

You may assume that the input list is non-empty.

You may assume that the input list contains the target.

#### test_00:

```js
const a = new Node("a");
const b = new Node("b");
const c = new Node("c");
const d = new Node("d");
const e = new Node("e");
const f = new Node("f");

a.next = b;
b.next = c;
c.next = d;
d.next = e;
e.next = f;

// a -> b -> c -> d -> e -> f

removeNode(a, "c");
// a -> b -> d -> e -> f
```

#### test_01:

```js
const x = new Node("x");
const y = new Node("y");
const z = new Node("z");

x.next = y;
y.next = z;

// x -> y -> z

removeNode(x, "z");
// x -> y
```

#### test_02:

```js
const q = new Node("q");

const s = new Node("s");

q.next = r;
r.next = s;

// q -> r -> s

removeNode(q, "q");

```

#### test_03:

const node1 = new Node("h");

node1.next = node2;

#### test_04:
