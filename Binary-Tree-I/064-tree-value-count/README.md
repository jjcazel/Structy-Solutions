# tree value count

Write a function, _treeValueCount_, that takes in the root of a binary tree and a target value.
The function should return the number of times that the target occurs in the tree.

#### test_00:

```js
const a = new Node(12);
const b = new Node(6);
const c = new Node(6);
const d = new Node(4);
const e = new Node(6);
const f = new Node(12);

a.left = b;
a.right = c;
b.left = d;
b.right = e;
c.right = f;

//      12
//    /   \
//   6     6
//  / \     \
// 4   6     12

treeValueCount(a,  6); // -> 3
```
